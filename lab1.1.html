1. Definitions:


a. HTML

This is the language for creating content of website by putting many elements with their "tags".

The content will be processed by browsers so that we can see and understand them.

 

b. CSS

The language is used to setting the style of web. 

It will present how the page's elements look like on the user interface. 


c. Javascript

The programming language are used to process the data at browsers before sending to server.


d. HTTP

It is protocol to transfer requests from client to server and responses from server to client. 


e. HTML5

The 5th version of HTML markup language has been popularly used until now (2017) 

It has new useful features and tags.

Websites using HTML5  allows playing multimedia is easily without browser plug-in. 


f. Dynamic HTML (DHTML)

It is the combination of HTML, CSS, Javascript to create a dynamic site that have animation 

and directly process some kind of data and request (such as Input text, selection, hover...)  at the browser
